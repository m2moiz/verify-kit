# Plan 04-03 — DB stack + Testcontainers

## Objective

Add the optional DB layer (gated on `has_backend AND has_db`): SQLAlchemy 2.x async +
asyncpg + Alembic with one example model + initial migration, plus a Testcontainers
Postgres fixture for integration tests. Also add `polyfactory` factory and
`dirty-equals` assertion helpers (API-13).

## Non-goals
- Schemathesis fuzz — Plan 04-07.
- Integrating DB routes into `app/api.py` — Plan 04-02 stays DB-free; this plan adds
  a `db-ping` style integration test only, not new HTTP routes.

## Must-haves (goal-backward)

1. `has_backend=true AND has_db=true` produces `app/db.py`, `app/schema.py`, `alembic/`,
   `alembic.ini`.
2. `has_backend=true AND has_db=false` produces ZERO of the above.
3. `has_backend=false` produces ZERO of the above (regardless of `has_db`).
4. `uv run alembic upgrade head` against a Testcontainers Postgres URL creates the
   `users` table successfully.
5. The `pg_container` pytest fixture caches across the test session and tears down
   correctly.
6. `uv run pytest tests/backend/test_db_integration.py -v` exits 0.

`alembic/env.py`:
```python
from __future__ import annotations
import asyncio
import os
from logging.config import fileConfig
from pathlib import Path
from alembic import context
from sqlalchemy import pool
from sqlalchemy.engine import Connection
from sqlalchemy.ext.asyncio import async_engine_from_config

from app.schema import Base

config = context.config
if config.config_file_name is not None:
    fileConfig(config.config_file_name)
target_metadata = Base.metadata

# DATABASE_URL substitution — pull from environment, NEVER from a relative Path.
# Alembic CLI is invoked from the scaffold root; cwd is set by `just` or test fixture.
url = os.environ.get("DATABASE_URL")
if url:
    config.set_main_option("sqlalchemy.url", url.replace("%", "%%"))


def do_run_migrations(connection: Connection) -> None:
    context.configure(connection=connection, target_metadata=target_metadata)
    with context.begin_transaction():
        context.run_migrations()


async def run_async_migrations() -> None:
    connectable = async_engine_from_config(
        config.get_section(config.config_ini_section, {}),
        prefix="sqlalchemy.",
        poolclass=pool.NullPool,
    )
    async with connectable.connect() as conn:
        await conn.run_sync(do_run_migrations)
    await connectable.dispose()


def run_migrations_online() -> None:
    asyncio.run(run_async_migrations())


run_migrations_online()
```

</objective>

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Public consumer's deployed FastAPI app → external clients | Untrusted; require_auth gates every route by default |
| Settings → env var VERIFYKIT_AUTH_TOKEN | Trusted (env var is operator-set); secret never logged |
| `/healthz` → docker/CI healthcheck | Intentionally unauthenticated for liveness probes |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-03 | Spoofing | Attacker sends forged X-VerifyKit-Token | mitigate | secrets.compare_digest (constant-time); 401 on mismatch |
| T-06-04 | Information Disclosure | Token logged in structured logs | mitigate | require_auth never logs `presented`; only logs "no token configured in dev" one-liner |
| T-06-05 | Tampering | Future refactor removes `dependencies=[Depends(require_auth)]` from main.py | mitigate | Task 3 polarity test grep-asserts the global dependency wiring + 401 behavior |
| T-06-06 | Denial of Service | Timing attack on token compare | mitigate | secrets.compare_digest is constant-time over equal-length encoded byte strings |
</threat_model>
